
# пакетная проверка страниц alkonorm.ru без отчетов

## цель
Улучшение индексируемости страниц в Яндексе и рост поискового трафика.

## порядок работы
- использовать только:
  - docs/seo/seo-batch-rules.md;
  - docs/seo/page-review-yandex-compact.md;
  - docs/seo/pages-status.md;

- брать первую страницу со статусом todo;
- выполнять полную проверку страницы;
- автоматически исправлять:
  - критичные;
  - важные;

- не исправлять желательно, если требуется существенная переработка;

- после исправлений выполнять контрольную проверку только измененных зон;

- если критичные проблемы устранены:
  - считать страницу обработанной;

- изменить статус страницы:
  - todo → done;

- перейти к следующей странице;

## вывод пользователю
- список обработанных URL;
- кратко:
  - что исправлено;
  - что осталось вручную;

## ограничения
- не создавать отчеты;
- не сохранять промежуточные файлы;
- не читать другие документы проекта без необходимости;
- не переписывать статью целиком;
- не добавлять неподтвержденные факты;
