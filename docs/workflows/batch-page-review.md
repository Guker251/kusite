# пакетная проверка страниц

## цель
Последовательно проверять страницы alkonorm.ru для улучшения индексируемости в Яндексе и роста поискового трафика.

## порядок работы

- прочитать docs/seo/pages-queue.md;
- прочитать docs/seo/pages-checked.md;
- определить первую непроверенную страницу;
- выполнить page-audit-yandex;
- исправить критичные и важные проблемы через page-fix-yandex;
- выполнить page-recheck-yandex;
- сформировать отчет;
- сохранить отчет в docs/seo/reports/;
- записать страницу как проверенную в docs/seo/pages-checked.md;
- перейти к следующей странице;

## формат ответа пользователю

- URL страницы;
- краткий итог;
- путь к отчету;
- что было исправлено в общем виде;
