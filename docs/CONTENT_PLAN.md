Слаг = путь относительно `articles/`.

### Принципы, по которым построен план
- Каждая статья закрывает один чёткий поисковый интент — пересечений нет
- Заголовок = основной ключ + уточнение (W-вопрос или выгода)
- Порядок внутри кластера — от большего охвата к меньшему
- Статьи «частично» релевантных запросов (норма за рулём, промилле, похмелье) выделены в отдельные URL, чтобы не размывать главные страницы хабов
- Хаб-страница остаётся агрегатором: содержит 3–4 секции по кластерам + ссылки на статьи
