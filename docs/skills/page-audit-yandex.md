
Назначение
проверка страницы на индексируемость в Яндексе;
выявление факторов, влияющих на трафик;
выявление контентных и структурных проблем;
Вход
URL страницы;
текст страницы;
целевой поисковый запрос (если есть);
тип страницы
статья / хаб / вспомогательная;
Выход

Список проблем:

проблема N;
уровень: критично / важно / желательно;
блок: индексируемость / SEO / контент / структура / поведенка;
описание;
причина (почему это влияет на Яндекс);
как исправить;
Алгоритм
1. Индексируемость (ядро)
есть ли страница в индексе;
корректен ли title;
есть ли H1;
нет ли дублей;
есть ли каноникал;
нет ли переоптимизации;
нет ли «тонкого контента»;
2. SEO под Яндекс
соответствие заголовка запросу;
наличие ключевых слов в:
title;
H1;
первых абзацах;
естественность текста;
отсутствие переспама;
наличие LSI-лексики;
3. Контент
раскрыта ли тема полностью;
есть ли практическая ценность;
есть ли структура:
списки;
подзаголовки;
нет ли воды;
4. Структура страницы
есть ли логика блоков;
есть ли перелинковка;
есть ли выход на другие статьи;
есть ли связь с хабом;
5. Поведенка (оценочно)
есть ли причины дочитать;
есть ли «зацепка» в начале;
есть ли логика удержания внимания;
есть ли следующий шаг;
Ограничения
не исправлять;
не переписывать;
не придумывать данные;
